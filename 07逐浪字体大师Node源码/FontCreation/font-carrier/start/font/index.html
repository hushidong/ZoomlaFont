你可以基于两种方式引用自己创作的字体：

方式一：在web网页、移动开发、NodeJS设计、小程序和公众号开发中使用

使用font-face声明字体-> 
@font-face {
  font-family: 'FontZ01';
  src: url('./fonts/FontZ01.eot'); /* IE9 */
  src: url('./fonts/FontZ01.eot?#iefix') format('embedded-opentype'), /* IE6-IE8 */
  url('./fonts/FontZ01.woff2') format('woff2'),
  url('./fonts/FontZ01.woff') format('woff'),
  url('./fonts/FontZ01.ttf') format('truetype'), /* chrome、firefox、opera、Safari, Android, iOS 4.2+*/
  url('./fonts/FontZ01.svg#FontZ01'); /* iOS 4.1- */
}
定义使用webfont的样式，只要将font-family属性写相同即可 
.FontZ01{
font-family: "FontZ01" !important;
font-size: 1.2rem;
font-style: normal;
-webkit-font-smoothing: antialiased;
-moz-osx-font-smoothing: grayscale;
}
你也可以直接将上面两段代码直接拷进CSS文件或网页的style区，最后只要在Html中为文字区域绑定相应的样式className即可： 
<i class="FontZ01">华夏民族五千年一朝复兴强国梦，大国工匠逐浪人追求卓越精益求精！</i>

方式二：在windows、linux、苹果mac系统桌面上使用

需要专业的字体编译工程师，为您发布成适合桌面系统的otf或ttf格式文件，这样你就可以安装，从而支持word、wps、Photoshop、Illustrator、CorelDrwa、AutoCad以及一切电脑离线环境中。
